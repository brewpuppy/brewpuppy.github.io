Release: BJCP Style Tree

Today I am launching the Beer Kingdom, a group of visualizations to help us better understand the various beer styles. The first of these visualizations is the BJCP Style Tree. Future visualizations will show the relationship between the different beer styles.

While I frequently find myself visiting the BJCP website for style information, I also frequently find myself wishing there was a better way to navigate the style data. The BJCP website is full of valuable information but the user interface is stuck in the 1990s. For this reason I have designed the BJCP Style Tree as a smoother, graphical way to access the style information. All the style information can be found here but in a way which takes advantage of today’s web technology.
